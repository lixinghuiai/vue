# vue2-web
vue2、vue-router2、webpack2框架

想从react转vue，可以跟着我这个项目走。  
教程网址：https://segmentfault.com/a/1190000009127162


![image](https://github.com/hyy1115/vue2-web/blob/master/public/demo.gif)

### 安装

```javascript
npm install
```

### 启动

```javascript
npm satrt
```

### 浏览器访问

```javascript
http://localhost:3055/
```
